## Welcome to Qovery

Qovery is a DevOps Automation Platform that empowers developers to deploy and manage their applications on their own cloud infrastructure. Built for modern cloud-native development, Qovery provides everything you need to run production workloads at scale.

## The Qovery Platform

Qovery consists of five integrated products that work together to provide a complete platform solution:

## Why Qovery?

### For Developers

- **Self-Service Platform**: Deploy without waiting for DevOps
- **GitOps Workflow**: Automatic deployments on every push
- **Preview Environments**: Test every pull request in isolation
- **Multi-Cloud Support**: Deploy on AWS, GCP, Azure, or on-premise

### For Platform Teams

- **Standardization**: Enforce best practices across all teams
- **Cost Visibility**: Track and optimize cloud spending
- **Security Compliance**: Meet SOC 2, GDPR, HIPAA requirements
- **Multi-Cluster Management**: Manage all your Kubernetes clusters

### Your Infrastructure, Your Control

Unlike traditional PaaS solutions, Qovery runs on **your own cloud infrastructure**:

- Full ownership and control
- No vendor lock-in
- Cost transparency
- Data sovereignty

## Platform Capabilities

### Provision: Multi-Cluster Management

- One-click Kubernetes cluster provisioning
- Multi-cloud and on-premise support
- Unified cluster management interface
- Automated infrastructure setup

### Deploy: Application Delivery

- Containerized applications and databases
- Automatic CI/CD pipeline generation
- Blue/green and canary deployments
- Rollback and disaster recovery

### Observe: Unified Observability

- Real-time metrics and logs
- Correlated application and infrastructure views
- Performance monitoring
- Alert management

### Optimize: Cost Control

- Real-time cost tracking
- Resource right-sizing recommendations
- Spot instance management
- Cost allocation by team/project

### Secure: Compliance & Security

- Role-based access control (RBAC)
- Secrets management
- Network policies and isolation
- SOC 2, GDPR, HIPAA compliance

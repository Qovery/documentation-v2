## Types

### Environment Variables

Non-sensitive configuration:

```bash
NODE_ENV=production
API_URL=https://api.example.com
```

### Secrets

Sensitive data (encrypted at rest):

```bash
DATABASE_PASSWORD=<encrypted>
API_KEY=<encrypted>
```

## Scopes

- **Built-in**: System variables (automatically injected)
- **Environment**: Available to all services in environment
- **Service**: Specific to one application/container

### Via Terraform

```hcl
environment_variables = [
  {
    key   = "NODE_ENV"
    value = "production"
  }
]

secrets = [
  {
    key   = "API_KEY"
    value = var.api_key
  }
]
```

## Variable Interpolation

Reference other services:

```
DATABASE_URL=${database.host}:${database.port}
```

# Advanced Settings Icons

This directory should contain SVG icons for cloud providers and service types referenced in the advanced settings documentation.

## Required Files:

### Cloud Provider Icons:
- `aws.svg` - Amazon Web Services logo
- `gcp.svg` - Google Cloud Platform logo
- `azure.svg` - Microsoft Azure logo
- `scaleway.svg` - Scaleway logo

### Service Type Icons:
- `application.svg` - Application service icon
- `container.svg` - Container service icon
- `cronjob.svg` - Cronjob service icon
- `job.svg` - Job service icon
- `helm.svg` - Helm service icon

### Database Icons:
- `database-container.svg` - Container database icon
- `database-managed.svg` - Managed database icon

## Note:
These icons need to be obtained from the Qovery design team or hub.qovery.com assets.

## Authentication

### Login

Authenticate with Qovery:

```bash
qovery auth
```

Using API token:

```bash
export QOVERY_CLI_ACCESS_TOKEN="your-token"
qovery auth
```

### Logout

```bash
qovery auth logout
```

## Context Management

### Set Context

Interactive context selection:

```bash
qovery context set
```

Set specific organization:

```bash
qovery context set --organization "My Org"
```

### List Contexts

```bash
qovery context list
```

### Current Context

```bash
qovery context current
```

## Project Commands

### List Projects

```bash
qovery project list
```

### Create Project

```bash
qovery project create --name "My Project"
```

### Delete Project

```bash
qovery project delete --project "project-id"
```

## Environment Commands

### List Environments

List all environments in current project:

```bash
qovery environment list
```

### Create Environment

```bash
qovery environment create \
  --name "staging" \
  --cluster "cluster-id" \
  --mode STAGING
```

### Deploy Environment

Deploy all services in an environment:

```bash
qovery environment deploy
```

### Stop Environment

```bash
qovery environment stop
```

### Delete Environment

```bash
qovery environment delete --environment "env-id"
```

### Environment Variables

List environment variables:

```bash
qovery environment env list
```

Add environment variable:

```bash
qovery environment env create \
  --key "API_KEY" \
  --value "secret-value" \
  --scope ENVIRONMENT
```

## Application Commands

### List Applications

```bash
qovery application list
```

### Deploy Application

Deploy specific application:

```bash
qovery application deploy --application "app-id"
```

Deploy with tag/commit:

```bash
qovery application deploy \
  --application "app-id" \
  --commit "abc123"
```

### Redeploy

Redeploy the same version:

```bash
qovery application redeploy --application "app-id"
```

### Stop Application

```bash
qovery application stop --application "app-id"
```

### Delete Application

```bash
qovery application delete --application "app-id"
```

### Update Application

Update application configuration:

```bash
qovery application update \
  --application "app-id" \
  --cpu 1000 \
  --memory 1024
```

## Log Commands

### Stream Logs

Stream real-time logs:

```bash
qovery log
```

Stream logs for specific application:

```bash
qovery log --application "app-id"
```

Follow logs with filter:

```bash
qovery log --application "app-id" --follow --filter "ERROR"
```

### Historical Logs

Get logs from last hour:

```bash
qovery log --application "app-id" --since 1h
```

Get logs from specific time range:

```bash
qovery log \
  --application "app-id" \
  --from "2024-01-01T00:00:00Z" \
  --to "2024-01-01T23:59:59Z"
```

## Shell Commands

### Execute Command

Run a command in application container:

```bash
qovery shell --application "app-id" --command "ls -la"
```

### Interactive Shell

Open interactive shell:

```bash
qovery shell --application "app-id"
```

### Port Forward

Forward a local port to application:

```bash
qovery port-forward \
  --application "app-id" \
  --port 8080:8080
```

## Global Flags

Most commands support these global flags:

```bash
--organization string   Organization ID or name
--project string        Project ID or name
--environment string    Environment ID or name
--application string    Application ID or name
--output string         Output format (table, json, yaml)
--debug                 Enable debug mode
--help                  Show help
```

## Examples

### Deploy Application to Staging

```bash
# Set context
qovery context set

# Select staging environment
qovery environment list

# Deploy application
qovery application deploy --application "my-api"

# Watch logs
qovery log --application "my-api" --follow
```

### Create and Deploy New Environment

```bash
# Create environment
qovery environment create \
  --name "feature-branch" \
  --cluster "prod-cluster" \
  --mode DEVELOPMENT

# Deploy all services
qovery environment deploy

# Check status
qovery status --watch
```

### Debug Application Issues

```bash
# Stream logs with error filter
qovery log --application "my-app" --filter "ERROR" --follow

# Open interactive shell
qovery shell --application "my-app"

# Check environment variables
qovery environment env list

# Redeploy application
qovery application redeploy --application "my-app"
```

### Manage Environment Variables

```bash
# List all variables
qovery environment env list

# Add new variable
qovery environment env create \
  --key "FEATURE_FLAG" \
  --value "enabled" \
  --scope ENVIRONMENT

# Update application variable
qovery application env create \
  --application "my-app" \
  --key "LOG_LEVEL" \
  --value "debug"

# Redeploy to apply changes
qovery application redeploy --application "my-app"
```

## Tips

<Tip>
Use `--output json` flag to parse CLI output in scripts and CI/CD pipelines.
</Tip>

<Tip>
Tab completion is available for bash and zsh. Run `qovery completion --help` for setup instructions.
</Tip>

<Warning>
Some destructive commands (delete, stop) may require confirmation. Use `--force` flag to skip prompts in automation.
</Warning>

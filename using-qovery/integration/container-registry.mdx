## Overview

Qovery integrates with all major container registries, allowing you to deploy pre-built container images or use them as cache for faster builds.

  </Step>

    Click "Add Container Registry" and select your provider
  </Step>

  <Step title="Use in Applications">
    Reference the registry when deploying container-based applications
  </Step>

## Use Cases

- **Deploy Pre-built Images**: Deploy containers built by external CI/CD pipelines
- **Build Cache**: Speed up builds by caching layers in your registry
- **Private Images**: Use private images from your organization's registry
- **Multi-Registry**: Connect multiple registries for different use cases

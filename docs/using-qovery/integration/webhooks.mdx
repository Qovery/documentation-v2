## Overview

Configure webhooks to receive real-time notifications about deployments and other Qovery events. Send notifications to Slack, Discord, custom endpoints, or incident management tools.

## Webhook Types

<AccordionGroup>
  <Accordion title="Standard Webhooks" icon="webhook">
    **Qovery JSON Format**

    Receive events in Qovery's proprietary JSON format:
    - Complete event details
    - Deployment metadata
    - Environment and service information
    - Custom payload processing

    Best for: Custom integrations and automation
  </Accordion>

  <Accordion title="Slack Webhooks" icon="slack">
    **Pre-formatted for Slack**

    Messages automatically formatted for Slack:
    - Rich message formatting
    - Direct to Slack channels
    - No transformation needed
    - Ready-to-use notifications

    Best for: Team chat notifications
  </Accordion>
</AccordionGroup>

## Supported Events

Configure webhooks to trigger on specific events:

- **DEPLOYMENT_STARTED** - When a deployment begins
- **DEPLOYMENT_SUCCESS** - Deployment completed successfully
- **DEPLOYMENT_FAILURE** - Deployment encountered errors
- **DEPLOYMENT_CANCELLED** - Deployment was cancelled

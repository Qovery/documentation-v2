## Built-in Monitoring

Qovery includes:

- **Metrics**: CPU, memory, network
- **Logs**: Centralized log aggregation
- **Events**: Deployment and lifecycle events

## Metrics

View in Console:

- Real-time CPU/memory usage
- Request rate and latency
- Error rates

## Logs

Stream logs:

```bash
qovery log --application my-app --follow
```

Filter logs:

```bash
qovery log --application my-app --filter "ERROR"
```

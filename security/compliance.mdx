## Compliance Standards

Qovery supports:

- **SOC 2 Type II**
- **GDPR**
- **HIPAA** (Enterprise)
- **ISO 27001**
- **PCI DSS** (with additional configuration)

## SOC 2

Qovery undergoes annual SOC 2 Type II audits:

- Security controls documented
- Third-party audited
- Reports available to enterprise customers

## GDPR

GDPR compliance features:

- **Data Residency**: Choose data location
- **Right to Erasure**: Delete user data on request
- **Data Portability**: Export all data
- **Privacy by Design**: Built-in privacy controls

### Data Processing Agreement (DPA)

Enterprise customers receive DPA covering:

- Data processing terms
- Sub-processor list
- Security measures
- Breach notification

## HIPAA

For healthcare applications (Enterprise):

- **Business Associate Agreement (BAA)**
- PHI encryption at rest and in transit
- Audit logging
- Access controls
- Breach notification procedures

## Audit Logging

Complete audit trail:

- All API calls logged
- User actions tracked
- Resource changes recorded
- Retention: 1 year (configurable)

### Export Audit Logs

```bash
qovery audit-log export \
  --from "2024-01-01" \
  --to "2024-12-31" \
  --format json
```

## Data Residency

Choose where your data is stored:

- **US**: us-east-1, us-west-2
- **EU**: eu-west-1, eu-central-1
- **Asia**: ap-southeast-1
- **Custom**: Any supported region

## Encryption

### Data at Rest

- Database encryption (AWS RDS, Cloud SQL)
- Volume encryption (EBS, Persistent Disks)
- S3 bucket encryption
- AES-256 encryption standard

### Data in Transit

- TLS 1.3 for all connections
- Certificate pinning
- HSTS enabled
- Perfect forward secrecy

## Vulnerability Management

- **Automated Scanning**: Container image scanning
- **Dependency Checks**: Check for known vulnerabilities
- **Patch Management**: Regular security updates
- **Incident Response**: 24/7 security team

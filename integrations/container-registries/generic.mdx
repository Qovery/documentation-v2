## Quick Setup

<Steps>
  <Step title="Get Registry Details">
    Obtain registry URL and credentials
  </Step>

  <Step title="Add in Qovery">
    Organization Settings → Container Registries → Add Registry
  </Step>

  <Step title="Select Generic">
    Choose "Generic Container Registry"
  </Step>

  <Step title="Configure">
    - Registry URL (with https://)
    - Username
    - Password or token
    - Optional: Skip TLS verification (not recommended)
  </Step>
</Steps>

## Quick Setup

### Registry URL

```
https://ghcr.io
```

### Image Format

```
ghcr.io/<owner>/<image-name>:<tag>
```

## Configuration Steps

<Steps>
  <Step title="Create Personal Access Token">
    GitHub Settings → Developer settings → Personal access tokens → Generate new token

    Required scopes:
    - `read:packages`
    - `write:packages` (if pushing)
  </Step>

  <Step title="Add Registry in Qovery">
    Organization Settings → Container Registries → Add Registry
  </Step>

  <Step title="Select GitHub CR">
    Choose "GitHub Container Registry"
  </Step>

  <Step title="Enter Credentials">
    - Username: Your GitHub username
    - Password: Personal Access Token
  </Step>

  <Step title="Save">
    Registry is now connected
  </Step>
</Steps>

## Supported Kubernetes Distributions

- Amazon EKS (self-managed)
- Google GKE (self-managed)
- Azure AKS (self-managed)
- DigitalOcean Kubernetes
- Linode LKE
- K3s, K0s, MicroK8s
- Rancher, OpenShift
- Self-hosted Kubernetes (kubeadm)
- On-premise clusters

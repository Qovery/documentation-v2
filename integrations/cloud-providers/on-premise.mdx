## On-Premise Considerations

- Self-managed Kubernetes cluster required
- Storage provisioner (Rook/Ceph, Longhorn, etc.)
- Load balancer solution (MetalLB, HAProxy, etc.)
- Ingress controller
- Network connectivity to Qovery API

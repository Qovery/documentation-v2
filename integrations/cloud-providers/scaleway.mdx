## Scaleway Services Integration

Qovery integrates with:
- **Kapsule**: Managed Kubernetes
- **Container Registry**: Private container registry
- **Managed Databases**: PostgreSQL, MySQL
- **Object Storage**: S3-compatible storage
- **Load Balancer**: Layer 4/7 load balancing

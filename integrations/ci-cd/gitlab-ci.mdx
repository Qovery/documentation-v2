## Quick Example

```yaml
deploy:
  stage: deploy
  image: alpine:latest
  script:
    - curl -s https://get.qovery.com | sh
    - export QOVERY_CLI_ACCESS_TOKEN=$QOVERY_TOKEN
    - qovery application deploy --application-name my-app
  only:
    - main
```

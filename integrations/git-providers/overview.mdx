## Overview

Connect your Git repositories to enable GitOps workflows with Qovery. Support for GitHub, GitLab, and Bitbucket allows you to deploy directly from your source code.

## Features

- **Auto-deploy**: Automatic deployments on push
- **Preview Environments**: Create environments for pull/merge requests
- **Multi-branch**: Deploy different branches to different environments
- **Webhook Integration**: Real-time deployment triggers
- **Repository Access**: Full access to all repositories when connected

## Overview

<img src="/images/logos/helm-icon.svg" alt="Helm" class="helm-icon" /> Qovery supports deploying Helm charts from various repository types, allowing you to use community charts or your own custom charts.

## Supported Repository Types

- **HTTP**: Standard Helm repositories
- **OCI Registries**: 
  - AWS ECR (public and private)
  - Google Artifact Registry
  - Azure Container Registry
  - GitHub Container Registry
  - GitLab Container Registry
  - Scaleway Container Registry
  - Docker Hub
  - Generic OCI registries

## Configuration

<Steps>
  <Step title="Add Helm Repository">
    Organization Settings → Helm Repositories → Add Repository
  </Step>

  <Step title="Select Type">
    Choose HTTP or OCI registry type
  </Step>

  <Step title="Configure">
    - Repository name
    - Repository URL
    - Credentials (for private repos)
  </Step>

  <Step title="Deploy Charts">
    Create Helm service in your environment using this repository
  </Step>
</Steps>

## Popular Helm Charts

- **Databases**: PostgreSQL, MySQL, MongoDB, Redis
- **Monitoring**: Prometheus, Grafana
- **CI/CD**: Jenkins, ArgoCD, Tekton
- **Messaging**: RabbitMQ, Kafka
- **Storage**: MinIO, Rook

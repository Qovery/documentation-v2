## Overview

Qovery provides built-in observability features and integrates with external monitoring tools.

## Built-in Observability

<CardGroup cols={2}>
  <Card title="Logs" icon="file-lines">
    - Real-time log streaming
    - 90-day retention (with package)
    - Pod-level filtering
    - Search and filter
  </Card>

  <Card title="Metrics" icon="chart-line">
    - CPU and memory usage
    - Network traffic
    - Request rates
    - Custom metrics
  </Card>

  <Card title="Events" icon="bell">
    - Kubernetes events
    - Qovery deployment events
    - Health check failures
    - Auto-scaling events
  </Card>

  <Card title="Health Checks" icon="heartbeat">
    - Liveness probes
    - Readiness probes
    - Application health status
  </Card>
</CardGroup>

## External Integrations

### Supported Tools

- **Datadog**: Full-stack monitoring and APM
- **Grafana**: Visualization and dashboards
- **Kubecost**: Kubernetes cost monitoring
- **CloudWatch**: AWS native monitoring (for EKS)

### Technology Stack

Qovery uses industry-standard tools:
- **Grafana Loki**: Log aggregation
- **Prometheus**: Metrics collection
- **Grafana**: Visualization

## Observability Package

Available on AWS, GCP, and Scaleway (Azure coming soon):
- 90-day log retention
- Extended metrics
- Custom dashboards
- Alert management

## Overview

External Secrets Operator is a Kubernetes operator that fetches secrets from external secret managers and creates Kubernetes secrets.

## Supported Backends

- AWS Secrets Manager
- GCP Secret Manager
- Azure Key Vault
- HashiCorp Vault
- And many more...

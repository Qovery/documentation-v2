## Features

- **Centralized Management**: Single source of truth for secrets
- **Automatic Sync**: Secrets synced to Qovery environments
- **Audit Logging**: Track secret access and changes
- **Rotation**: Automatic secret rotation support
- **Multi-environment**: Different secrets per environment

## Best Practices

- Use secret managers for production secrets
- Enable audit logging
- Implement least-privilege access
- Rotate secrets regularly
- Never commit secrets to Git
